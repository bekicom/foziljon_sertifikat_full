# foziljon_sertifikat_full

Full certificate project with React frontend and Express/MongoDB backend.

## Folders

- `frontend_edu` - React frontend
- `backend_cert` - Express backend

## Local Backend

Create `backend_cert/.env` from `backend_cert/.env.example`, then run:

```bash
cd backend_cert
npm install
npm run seed:admin
npm run dev
```

Default seeded admin:

```text
admin / 0000
```

## Local Frontend

```bash
cd frontend_edu
npm install
npm start
```
